# V68 – Transform Gizmo + Group Authoring

V68 pushes the in-engine builder toward non-coder usability.

## Added in this version
- multi-select editor selection
- Ctrl+drag box selection
- group-aware translate / rotate / scale gizmo
- W / E / R gizmo hotkeys
- Ctrl+D duplicate selected group
- M mirror selected group across X
- Delete remove selected group
- on-screen box-select overlay

## Workflow
1. Use left click to pick a single entity.
2. Hold Ctrl and click additional entities to add to the working set.
3. Hold Ctrl and drag to box select in screen space.
4. Press W / E / R to switch gizmo mode.
5. Use J/L, I/K, U/O to manipulate the current group.
6. Press Ctrl+D to duplicate the selection as a new group.
7. Press M to mirror the group.
8. Press Delete to remove the selected group.

## Why this matters
This version shifts the editor from one-off object nudging into real blockout authoring. It is the right bridge between primitive stamping and a future kit palette.
