# 3D_GAME_ENGINE v21 — Real Exploration Gameplay

This version turns the visible world into a playable exploration slice.

## Added
- in-world interaction prompt
- quest acceptance and completion loop
- gatherable herb pickups in the forest
- reward turn-in with gold and item reward
- travel points that move you between visible regions
- simple NPC interaction based on proximity

## Controls
- WASD move
- Mouse look
- E interact / gather / talk / travel
- 1 / 2 / 3 direct region switch
- F5 save, F9 load
