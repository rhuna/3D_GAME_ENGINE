# 3D_GAME_ENGINE v20 — Visible World Rendering + Region Layouts

This version makes the open-world branch visible in 3D.

## Added
- region layout database with visible props
- world renderer
- camera + player movement in visible 3D regions
- NPC placeholder rendering
- gather point and travel point rendering
- region switching changes what you see

## Controls
- WASD move
- Space / Left Ctrl move vertically
- Mouse move look
- 1 / 2 / 3 switch regions
- F5 save, F9 load
