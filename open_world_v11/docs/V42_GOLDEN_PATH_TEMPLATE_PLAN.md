# V42 Starter Template / Golden Path Plan

This version defines the first reusable playable slice for the boilerplate.

The golden path includes:
- one outdoor starter region
- one enterable building
- one merchant NPC
- one gather node
- one basic crafting recipe
- one simple hostile encounter
- one save point
- one quest handoff stub

Purpose:
- prove the current runtime can support a compact vertical loop
- establish naming conventions for content authors
- give future versions a sample that can be duplicated instead of rebuilt from scratch
