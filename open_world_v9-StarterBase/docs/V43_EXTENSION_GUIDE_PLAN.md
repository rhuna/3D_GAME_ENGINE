# V43 Documentation and Extension Pass

## Purpose
V43 is the point where the project stops being only a private scaffold and starts acting like a reusable starter framework.

## Goals
- document the current runtime path
- document how to add starter content without modifying engine code first
- establish extension points for regions, prefabs, NPCs, merchants, gather nodes, quests, factions, and companions
- define the expected folder layout for future game content

## Extension surfaces
1. Prefabs
2. Scene files
3. Region runtime markers
4. Dialogue and quest definitions
5. Merchant inventories
6. Gather/crafting definitions
7. Persistent world markers
8. Faction standing data
9. Companion definitions

## Boilerplate-ready exit criteria
- starter template content can be duplicated and renamed to create a second sample zone
- adding a merchant, gather node, save point, and crafting station follows documented steps
- folder conventions are stable enough that later versions can focus on content instead of architecture
