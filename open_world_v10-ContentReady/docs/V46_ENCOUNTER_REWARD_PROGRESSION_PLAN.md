# V46 Encounter, Reward, and Progression Deepening Plan

V46 expands the Starter Vale vertical slice from a single authored loop into a more game-like progression lane.

## Goals
- Add stronger encounter definitions with escalation hooks.
- Add reusable reward bundle definitions.
- Add basic progression milestone scaffolding.
- Connect faction standing changes to slice outcomes.

## Intended loop
1. Talk to Mara.
2. Gather herbs.
3. Fight or avoid wolves in the brush.
4. Craft field medicine.
5. Return for reward.
6. Update progression milestone and herbalist standing.

## Follow-up targets
- V47: multi-step authored quest chains.
- V48: second region and repeatable content pack pattern.
