# Harden Starter Base

Use this pass when the framework is feature-complete enough and the priority shifts to consistency.

## Steps
1. Keep one recommended runtime path.
2. Prefer one spawn/content entry point.
3. Prefer one quest/runtime state model.
4. Keep starter content packs aligned to the current manifests.
5. Remove or quarantine temporary compatibility wrappers.
6. Update version notes and extension docs after each cleanup step.
