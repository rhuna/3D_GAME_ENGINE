# V80 Economy Authoring Suite

V80 adds a low-code economy authoring layer to the GameBuilderPanel.

## Added
- New **Trade** tab
- Item graph generation to `assets/itemgraphs/*.itemgraph`
- Economy bundle generation to `assets/links/*_economy.links`

## Updated
- Item authoring now uses editable type/value/heal fields
- Merchant authoring now uses editable stock count and stock price
- Loot authoring now uses editable drop count and drop weight

## Workflow
1. Open Trade
2. Enter item, merchant, and loot values
3. Create Item / Item Graph / Merchant / Loot Table
4. Create Economy Bundle
