# V81 Notes — Region Painter / Brush Authoring

## Goal
V81 reduces the last major world-building friction point by adding low-code brush preset and paint bundle authoring to the builder.

## Main additions
- New **Brush** tab in `GameBuilderPanel`
- Brush preset authoring for foliage / prop stamping
- Paint bundle export for region-level brush setup
- New output folders and files:
  - `assets/brushes/*.brush`
  - `assets/world/*.paint`

## Why V81 matters
V72–V80 made gameplay content increasingly low-code.
V81 does the same for broad world decoration and density control so game creation needs fewer one-by-one placements.
