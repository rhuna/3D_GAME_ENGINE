# V47 Quest Chain Progression Plan

Goal: expand the starter vale slice from a single loop into a reusable authored quest chain.

## Focus
- multi-step quest flow
- reusable objective bundles
- branching rewards
- follow-up dialogue
- progression flags that can gate later content

## Starter Vale chain
1. Talk to Mara
2. Gather herbs
3. Craft field medicine
4. Return to Mara
5. Optional follow-up with Guard Tomas

## Boilerplate value
This version moves the project closer to content-ready by proving a repeatable authored quest-chain pattern.
