# Build a Vertical Slice

Use this checklist when turning a content pack into a playable slice.

## Minimum required content
- region scene
- one interior scene
- one merchant NPC
- one gather node type
- one crafting recipe
- one starter quest
- one hostile encounter
- one save point

## Recommended validation
- load into the region
- interact with at least one NPC
- collect at least one item
- complete one quest objective
- save and reload the world state
