# Add a New Content Pack

A content pack groups authored data for a region or gameplay slice.

Recommended layout:
- `assets/content/packs/<pack_id>/pack.json`
- `assets/content/packs/<pack_id>/npcs/*.json`
- `assets/content/packs/<pack_id>/quests/*.json`
- `assets/content/packs/<pack_id>/encounters/*.json`
- `assets/content/packs/<pack_id>/loot/*.json`

Content packs should be additive and data-driven.
