# 3D_GAME_ENGINE v22 — Third-Person Exploration Combat Layer

This version upgrades the visible-world branch into a basic action-RPG slice.

## Added
- third-person follow camera
- player movement with facing
- sprint and jump
- enemy spawn points and placeholder enemies
- melee attack with hit range
- enemy chase/attack behavior
- health / damage / defeat / respawn loop
- encounter clear tracking per region
- quest, gathering, travel, and NPC interaction kept active

## Controls
- WASD move
- Mouse move camera
- Shift sprint
- Space jump
- Left Click attack
- E interact
- 1 / 2 / 3 switch regions
- F5 / F9 save / load
