# Add a New Region

1. Duplicate an existing scene in `assets/scenes/`.
2. Register a new region id in your region data source.
3. Add travel markers that link into and out of the region.
4. Add any starter prefabs such as NPCs, save points, and gather nodes.
5. Add persistence markers if the region should remember state changes.
6. Verify the region can be entered from the golden path template.
