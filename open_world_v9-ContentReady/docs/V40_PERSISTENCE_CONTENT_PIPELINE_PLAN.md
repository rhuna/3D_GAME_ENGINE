# V40 Persistence + Content Pipeline Plan

This milestone pushes the project toward boilerplate readiness by standardizing persistent world identities and content-authored runtime markers.

## Goals
- Persistent IDs for authored runtime entities
- Serializable world deltas for harvested / looted / defeated / activated state
- Cleaner authored marker flow for travel, save, gather, NPC and encounter content
- A stable foundation for future content-ready vertical slices
